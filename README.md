# Balzac

A Clojure library to read epub/mobi metadata.

## Dependency for Leiningen Projects

To add Balzac as a dependency to your Leiningen project, use:

[balzac "0.1.0"]

## Usage

```clojure
(def is (input-stream "/path/to/file"))
(book is)
```
